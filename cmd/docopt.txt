Usage:
    dapple config
    dapple init
    dapple install [--save] [<package>]
    dapple build [options]
    dapple chain
    dapple step
    dapple test [options] [--skip-build]
    dapple new-test <class>

Options:
    -e ENVIRONMENT, --environment ENVIRONMENT [default: evm]
    -c DAPPLERC, --config DAPPLERC
